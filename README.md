# Telegram Bot для учета материальных ценностей

Бот для учета материальных ценностей с интеграцией Google Sheets.

## Структура проекта

```
bot-accounting/
├── src/                    # Исходный код
│   ├── handlers/          # Обработчики команд и сообщений
│   ├── keyboards/         # Клавиатуры
│   ├── services/          # Сервисы (БД, Sheets, QR)
│   ├── middlewares/       # Middleware
│   └── utils/            # Утилиты
├── data/                  # База данных SQLite
├── logs/                  # Логи приложения
├── credentials/          # Учетные данные (service_account.json)
└── docker/               # Docker конфигурация
```
